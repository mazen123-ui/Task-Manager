# Task Manager App

A simple and efficient **Task Manager** web application built with **Vanilla JavaScript**, HTML, and CSS.  
The app allows users to manage daily tasks with full CRUD functionality and persistent storage using **LocalStorage**.

This project focuses on clean logic, state-driven UI, and real-world frontend fundamentals.
